Die Anfrage wird identifiziert durch den Eventnamen START_VERSAND_SDAE. Zusätzlich ist eine eindeutige ID prozessId aus dem Backend mit zu übergeben, mit der die spätere Antwort vom Marktpartner wieder an das Backend übergeben werden kann.
